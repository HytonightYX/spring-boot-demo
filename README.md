# 基于spring-boot 简单的书单管理系统
书单管理系统

# 环境: 

Java 8 MySQL 5.7

# 技术栈:

基于 springBoot 2.1.1 

持久层: Spring-Data JPA

前端部分: thymeleaf 3 + bootstrap 3

构建工具: maven

版本详情见pom.xml
